# Implementation Plan: UI Design System

**Branch**: `005-ui-redesign` | **Date**: 2026-06-08 | **Spec**: [spec.md](spec.md)

**Input**: Feature specification from `specs/005-ui-redesign/spec.md`

## Summary

Replace the scattered, ad-hoc styling across four UI files with a single design system: a centralised `theme.py` module holding all colour tokens and CSS generators, plus a `.streamlit/config.toml` for Streamlit's built-in theme layer. The result is a slate-navy professional palette with consistent badge styles, tab states, typography scale, and status messages — with no new Python packages.

## Key Design Decisions

### 1. Colour Palette — 8 Slate-Navy Tokens

All colours derive from the same cool-neutral family:

| Token | Hex | Usage |
|-------|-----|-------|
| `PRIMARY` | `#1e3a5f` | Page title, section headings |
| `INTERACTIVE` | `#2563eb` | Active tab indicator, buttons, chart line |
| `BUY` | `#16a34a` | BUY badge foreground + light green tint background |
| `SELL` | `#dc2626` | SELL badge foreground + light red tint background |
| `HOLD` | `#b45309` | HOLD badge foreground + amber tint background |
| `NEUTRAL_LIGHT` | `#f8fafc` | Page background, tab strip |
| `NEUTRAL_MID` | `#e2e8f0` | Inactive tab background, dividers |
| `NEUTRAL_DARK` | `#0f172a` | Body text, active tab foreground |

### 2. Two-Layer Theming

- **Layer 1 — config.toml**: Sets Streamlit's `primaryColor` (`#2563eb`), `backgroundColor` (`#f8fafc`), `secondaryBackgroundColor` (`#e2e8f0`), `textColor` (`#0f172a`), `font = "sans"`. Propagates automatically to all built-in widgets.
- **Layer 2 — injected CSS**: Handles what config.toml can't reach: tab shapes, badge cells, type scale overrides, status message uniformity. Called once via `inject_global_css()` at the start of `render_dashboard()`.

### 3. Signal Badge via Pandas Styler

Replace text-colour-only signal styling with foreground + background cell styling:
- BUY: `background-color: #dcfce7; color: #16a34a; font-weight: bold`
- SELL: `background-color: #fee2e2; color: #dc2626; font-weight: bold`
- HOLD: `background-color: #fef3c7; color: #b45309; font-weight: bold`

Row-click interaction (`on_select="rerun"`) is preserved — `st.dataframe` with Styler supports both.

### 4. Typography Scale (CSS-injected)

Four levels applied via global CSS:
- **Page title** (`h1`): `1.75rem / 700 / PRIMARY`
- **Section heading** (`h2`, `h3`): `1.1rem / 600 / PRIMARY`
- **Body** (default `p`, `div`): `0.9rem / 400 / NEUTRAL_DARK`
- **Caption** (`.stCaption`): `0.75rem / 400 / #64748b` (slate-400 — within palette family)

### 5. No New Dependencies

Zero new packages. All features use existing Streamlit, Pandas, Plotly capabilities.
